center-anything

Css tricks for centering anything

tricks

Assume we have a parent element and a child element, both are block level.

1. Zero Absolute Position

.parent {
    position: relative;
}
.child {
    position: absolute;
    margin: auto;
    top: 0;
    right: 0;
    bottom: 0;
    left: 0;
}

demos/zero-absolute-position.html

2. Table Cell Display

.parent {
  display: table-cell;
  vertical-align: middle;
}
.child {
  margin: auto;
}

demos/tabel-cell-display.html

3. Flex Display

.parent {
  display: flex;
  align-items: center;
  justify-content: center;
}
